# robinfadel.github.io

This repository contains a simple single-file GitHub Pages site (index.html) for the user `robinfadel`.

How to publish locally:

1. Create a repository named `robinfadel.github.io` under your GitHub account.
2. Copy the `index.html` into the repository root.
3. Commit & push:

```bash
git init
git add index.html README.md
git commit -m "Add initial GitHub Pages site"
git branch -M main
git remote add origin https://github.com/robinfadel/robinfadel.github.io.git
git push -u origin main
```

4. Visit https://robinfadel.github.io (may take a minute to become available).
